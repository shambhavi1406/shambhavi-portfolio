# Shambhavi Portfolio Website

A modern personal portfolio website with an AI-powered backend integration built using HTML, CSS, JavaScript, Node.js, Express, and MERN stack concepts.

## Features

- Responsive modern portfolio design
- About Me section
- Skills section
- Backend Development (MERN Stack)
- Projects showcase
- Contact section
- AI Assistant backend integration
- Clean UI and smooth layout
- GitHub-ready project structure

---

## Tech Stack

### Frontend
- HTML5
- CSS3
- JavaScript

### Backend
- Node.js
- Express.js
- REST API basics
- AI API integration
